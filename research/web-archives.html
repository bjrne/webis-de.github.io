Synopsis

The World Wide Web is the single largest repository of digital culture and knowledge. Given both ubiquitous availability and the constant stream of new and noteworthy content competing for the attention of web users, only few take notice of the fact that, at any time, reams of "old" content disappears—because web pages as well as entire websites are updated, restructured, and deleted. This project aims to improve the robustness of web archiving methods as well as to analyze web archives at scale.
